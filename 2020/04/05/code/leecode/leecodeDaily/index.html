42接雨水

leecode42接雨水

方法：层次遍历，按高度，一层一层计算空的位置。

双指针，left，right，在left-right区间内，根据当前的高度Min（height[left] , height[right]）计算这一层有多少个格水。

单调栈：

甜姨题解详情

最理想的揭发：根据各自的高度，直接计算该柱能够承载的水。

要知道每一部分确定的水，需要知道最后能够形成的水平高度，即该柱左右比它高的柱子的高度。但当前找到的最近的柱子不一定是最终形成水的柱子，例如高度5 2 3 5，针对2，3和5不是最后解，只是计算了到其的层数。在这种情况下，是按层次计算的。

单调递减栈，当前高度与栈顶比较，栈存下标，根据栈存的下标进行高度比较

  1. 当前列
    1.1 temp < peek, 进入2

    1.2 temp = peek ， pop，直到不相等

    1.3 temp > peek ，针对栈顶的列，可以根据 当前与次栈顶的列可以计算二者组成凹槽积蓄的水。然后将栈顶出栈，当前列不变，进入1

    若不存在次栈顶元素，说明当前栈顶列不会容纳水

  2. 入栈，列++，进入1
