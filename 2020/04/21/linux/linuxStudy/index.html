Linux 学习笔记

常用命令

  • pwd：显示当前路径
  • ls：列出当前目录下文件
    • ls -l：以列表形式出现
  • mkdir
  • rmdir：删除空文件夹
  • rm ：recurse递归删除
    • rm -r +name：有询问，是否删除
    • rm -rf name：强制全部删除
  • cp: copy文件 cp 1 2
  • mv: move文件 mv 1 2
  • vi：vim，文本编辑器
  • clear ：清屏
  • find：find + 路径 + name ： 查找文件
  • whereis ：whereis name 东西在哪

文件

  • ln “建立连接，硬/软连接，
    • ln a b
  • touch：生成新文件，修改文件的时间
  • find ：查找文件 find + 路径 + name
  • tree：显示目录树
  • wc：统计文件的行数，字数，字符数

    • cat：显示文件信息
    • tac：反序显示文件信息
    • head ：可选项，查看前多少行
    • tail：可选项，查看后多少行
  • grep：查找字符所在文件的位置— grep 字符 文件名
  • col：过滤字符

文件权限

  • ls - l :前面的是文件的权限

    • r：read
    • w：write
    • -：没有任何权限
    • x：执行权限

    三组，第一组，文件的所有者

    第二组：文件所有者的同组

    第三组，剩下的其他人

  • chmod：修改文件权限,默认u

    • chmod [u/g/o] +/- 权限名 文件名
    • chmod 755 文件名 （三组权限，9位表示）

  • chown： chown username 文件名：修改文件默认的用户

  • stat：显示文件信息

  • whoami：显示当前用户身份

目录意义

  • /：根分区

  • SWAP：虚拟内存分区

  • /boot：启动程序

  • /urs：

  • /bin：系统所需要的命令位于此目录，文件都是可执行的

  • /dev：设备文件存储目录，声卡，磁盘等

  • /etc：系统配置文件所在地，

  • /home：普通用户默认存放目录

  • lib：库文件存放目录

linux分区的命名 /dev/xxyN

/dev/：设备文件所在的目录名，分区在硬盘上，硬盘是设备，硬盘上所有可能的分区。

xx：分区的前两个字母表明1分区所在的设备的类型，hd（IDE磁盘），sd（SCSI磁盘）

y：表明分区所在的设备 a第一个，b第二个

N：代表分区，主分区（扩展分区） 1-4，逻辑分区从5开始

hda3：第一个IDE磁盘上的第三个分区

磁盘分区与挂载点 /mnt

boot顺序

  1. load bios
  2. read MBR‘s config to find out the OS
  3. load the kernel of the OS
  4. init process starts
  5. execute /etc/rc .d/sysinit runcommond demon
  6. start other modules(etc/modules.conf)
  7. execute the run level scripts
  8. execute /etc/rc.d/rc.local
  9. execute /bin/login
  10. shell started
