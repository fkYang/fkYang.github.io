双重检查

不使用volatile变量，会使得 instance = new Singleton();对象的过程能够被重排序，变成分配空间，赋值，然后初始化的顺序。这个过程其他线程能够看到赋值。但是看到的赋值并没有完成初始化。

解决方法：

  • 禁止重排序：双重检查，volatile

  • 让重排序其他线程不可见：静态内部类方式
